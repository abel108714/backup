





SELECT 表1.欄位a, 表2.欄位b
FROM 表1
(INNER/LEFT/RIGHT/FULL) JOIN 表2
(ON) 表1.連結欄位=表2.連結欄位


INNER JOIN 兩邊都為key 沒有空值
LEFT JOIN 左邊為key 右邊有空值
RIGHT JOIN 右邊為key 左邊有空值
FULL JOIN 兩邊都為key 有空值

CROSS JOIN 兩邊都為key 
NATURAL JOIN 等同 INNER JOIN 但不用ON




select Name, 
case Answer
  when 1 then '喜歡'
  when 2 then '不喜歡'
  when 3 then '還OK'
END
FROM questionnaire;

或是：
select Name, 
case
  when Answer=1 then '喜歡'
  when Answer=2 then '不喜歡'
  when Answer=3 then '還OK'
END
AS Answer
FROM questionnaire;

TRIM 函數是用來移除掉一個字串中的字頭或字尾。
最常見的用途是移除字首或字尾的空白
TRIM ( [ [位置] [要移除的字串] FROM ] 字串): [位置] 的可能值為 LEADING (起頭), TRAILING (結尾), or BOTH (起頭及結尾)。 這個函數將把 [要移除的字串] 從字串的起頭、結尾，或是起頭及結尾移除。如果我們沒有列出 [要移除的字串] 是什麼的話，那空白就會被移除。 
LTRIM (字串): 將所有字串起頭的空白移除。 
RTRIM (字串): 將所有字串結尾的空白移除。
